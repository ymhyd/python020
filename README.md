### 五、项目介绍

项目编号：python020

一个集可视化数据分析，信息管理等功能于一身的可供学生进行自我管理以及学校管理学生信息的学生信息管理系统。

学生信息管理系统涵盖学生、班级、课程、成绩、活动、用户及日志管理功能，实现对学生在校多维度信息的统筹管控 。
